\## Project Description



This project implements a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL. The ALU performs arithmetic and logical operations based on a 3-bit select input.



\---



\## Operations Supported



| Select | Operation |

|---------|-----------|

|000|Addition|

|001|Subtraction|

|010|AND|

|011|OR|

|100|XOR|

|101|NOT|

|110|Left Shift|

|111|Right Shift|
